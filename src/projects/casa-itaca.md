# Casa Itaca Vacanze 🏖️

The story behind this project lies on a good friend of mine, that owns this business that she rented on the typical vacation booking sites. Instead of paying high commissions to these platforms, she asked me for help to create her own website to rent her properties directly without intermediators.

## Main Functionality

- **Homepage**
- **Apartment Listings**
- **Gallery**
- **Pricing API**
- **Calendar availability API**
- **Booking API**
- **Email contact section**

## Booking Request Form

The site features an enquiry/request form (not a live availability booking engine). Users can submit:
- Check-in/check-out dates
- Guest count
- Apartment preference
- Special requests

## Technologies Used

- **Frontend**: React and Tailwind CSS
- **Backend**: Node.js and Express
- **Database**: Supabase (PostgreSQL)
- **Email API**: EmailJS
- **Hosting**: Vercel
