This application helps organise everything required to plan a wedding by keeping tasks, deadlines, budgets, guests and vendors in one place.

## Target Users
- Couples planning their own wedding
- Professional wedding planners
- Wedding vendors collaborating on an event

## Main Functionality
- **Task management**: Create tasks, assign due dates, mark progress, track status
- **Budget tracking**: Monitor wedding expenses and budgets
- **Guest management**: Keep guest lists and related information
- **Table setup**: Create tables and assign guests to tables
- **Vendor management**: Track photographers, caterers, florists, etc.
- **Timeline**: Organise the sequence of events leading up to the wedding
- **Photography**: Organise the photograph group shots and family groups, so you don't miss anybody out

## Architecture
- **Frontend**: React + TypeScript (Responsive interface for desktop and mobile)
- **Backend**: Supabase (Database, Authentication through RBAC)
- **Hosting**: Static frontend with Supabase providing backend services
